# 💰 Smart Expense Manager 

Smart Expense Manager ek powerful aur user-friendly web application hai jo users ko unke daily kharchon (expenses) ko track karne aur analyze karne mein madad karti hai. Ismein humne **Python (Flask)** ka use karke backend banaya hai aur **Chart.js** se data visualization add kiya hai.

---

## 🌟 Key Features

- **🔒 Secure Login:** Mobile number-based authentication system (Mock OTP 1234).
- **⚡ Real-time Updates:** Naya expense add karte hi charts aur total amount automatically update hote hain.
- **📱 Responsive Design:** Horizontal layout jo desktop aur mobile dono par sahi dikhta hai.
- **📂 Data Management:** CRUD (Create, Read, Update, Delete) functionality ke saath data file system mein save hota hai.

---

## 🛠️ Tech Stack Used

- **Backend:** Python 3.x, Flask (Framework)
- **Frontend:** HTML5, CSS3 (Flexbox Layout), JavaScript (ES6)
- **Data Storage:** YAML (Configuration) & TXT (Database)
